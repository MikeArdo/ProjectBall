# ProjectBall
Arcade endless game creato in Unity 3D

Evita gli ostacoli e segna il miglior punteggio!

Tocca lo schermo per cambiare la posizione della palla.
Attento a non farti ingannare dalla prospettiva!

(Gioco creato per hobby a 14 anni)
